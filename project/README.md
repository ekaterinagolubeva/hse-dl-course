## 🎨 Концепция проекта

### Основная идея

Проект **"От Атлантики до Авиньона: диалог эпох"** представляет собой художественный эксперимент, соединяющий два противоположных полюса в истории искусства:

- **Уинслоу Гомер** (1836-1910) — американский художник-реалист, мастер морских пейзажей и сцен из жизни простых людей
- **Пабло Пикассо** (1881-1973) — испанский художник, основоположник кубизма, революционер в мире искусства

### Цель проекта

Создать визуальный диалог между двумя эпохами и стилями, показав, как одни и те же современные темы могли бы быть интерпретированы через призму классического американского реализма XIX века и авангардного европейского кубизма XX века.

### Философия

Проект исследует вопрос: **"Как искусство прошлого видело бы наше настоящее?"**

Через обученные на работах Гомера и Пикассо модели Stable Diffusion мы создаём альтернативные визуальные реальности, где современные объекты, технологии и социальные явления получают художественное осмысление через стилистику прошлых эпох.

## 📚 Исходные данные

### Источник данных

Датасет собран из открытой коллекции **National Gallery of Art** (Вашингтон, США) через их официальный Open Data API.

### Параметры выборки

- **Авторы**: Winslow Homer, Pablo Picasso
- **Тип произведений**: Только живопись (Painting)
- **Исключены**: наброски, эскизы, рисунки, технические материалы
- **Формат**: JPG, 360px (высота/ширина в зависимости от ориентации)
- **Количество изображений**: 31 произведение

### Распределение по авторам

Датасет включает работы обоих художников, представляющие различные периоды их творчества:

**Уинслоу Гомер:**
- Морские пейзажи (главная тема)
- Сцены рыболовства
- Природные стихии
- Жанровые сцены

**Пабло Пикассо:**
- Работы кубистического периода
- Портреты
- Натюрморты
- Абстрактные композиции

## 🎭 Тематические серии

Проект включает **4 тематических серии**, каждая из которых представляет парные работы в стилях Гомера и Пикассо:

### 1. **Море и стихия** 🌊

**Концепция:** Исследование морской темы — ключевой для Гомера и менее характерной для Пикассо.

**Промпты для генерации:**
- *Стиль Гомера*: "Бурное море во время шторма, высокие волны, драматичное небо, реалистичная живопись"
- *Стиль Пикассо*: "Море в кубистическом стиле, геометрические волны, абстрактные формы воды"

**Цель:** Показать контраст между романтическим реализмом Гомера и геометрической абстракцией Пикассо.

---

### 2. **Городская жизнь** 🏙️

**Концепция:** Современный городской пейзаж глазами художников прошлого.

**Промпты для генерации:**
- *Стиль Гомера*: "Современный город с небоскрёбами, реалистичная атмосферная живопись, драматичное освещение"
- *Стиль Пикассо*: "Городской пейзаж в кубистическом стиле, геометрические здания, фрагментированные формы"

**Цель:** Визуализировать урбанистическую среду XXI века через оптику XIX и XX веков.

---

### 3. **Портреты современности** 👤

**Концепция:** Люди с современными гаджетами и в современных ситуациях.

**Промпты для генерации:**
- *Стиль Гомера*: "Человек со смартфоном, реалистичный портрет, естественное освещение"
- *Стиль Пикассо*: "Портрет человека с телефоном в кубистическом стиле, множественные перспективы"

**Цель:** Создать анахронистические образы, соединяющие современные технологии и классическую живопись.

---

### 4. **Природа и индустрия** 🏭

**Концепция:** Экологический конфликт между природой и промышленностью.

**Промпты для генерации:**
- *Стиль Гомера*: "Индустриальный пейзаж с заводами и дымом, реалистичная живопись"
- *Стиль Пикассо*: "Фабрика и природа в кубистическом стиле, геометрические формы, контраст"

**Цель:** Показать актуальную экологическую проблематику через художественные стили прошлого.

## ⚙️ Технические детали

### Инструменты и технологии

**Сбор данных:**
- Python 3.12
- Pandas для обработки метаданных
- Requests для загрузки изображений
- ThreadPoolExecutor для параллельной загрузки

**Обучение модели:**
- Stable Diffusion (базовая модель: SD 1.5 или SDXL)
- LoRA (Low-Rank Adaptation) для эффективного файн-тюнинга
- Dreambooth / Textual Inversion (на выбор)

**Вычислительные ресурсы:**
- GPU: NVIDIA с минимум 10GB VRAM (или Google Colab Pro)
- Время обучения: ~2-4 часа на одну модель

### Параметры обучения

**Предварительная обработка:**
- Кропирование в соотношении 1:1 (512×512 или 768×768)
- Цветовая коррекция и нормализация
- Аугментация: flip, rotate, brightness adjustment

**Гиперпараметры:**
- Learning rate: 1e-4 — 5e-4
- Batch size: 1-2 (в зависимости от VRAM)
- Training steps: 1000-2000 для каждой модели
- Optimizer: AdamW

**Триггерные слова:**
- Для Гомера: `[sks_homer]`, `homer_style`, `winslow_homer_painting`
- Для Пикассо: `[sks_picasso]`, `picasso_style`, `cubist_painting`

### Процесс генерации

**Базовые параметры:**
- Sampler: Euler A или DPM++ 2M Karras
- Steps: 30-50
- CFG Scale: 7-10
- Seed: Фиксированный для воспроизводимости

**Промпт-инженеринг:**
- Использование весов для усиления стилистических элементов
- Negative prompts для исключения нежелательных элементов
- Композитные промпты для сложных сцен

## 📝 Заключение

Проект **"От Атлантики до Авиньона: диалог эпох"** представляет собой попытку создать визуальный мост между прошлым и настоящим, между реализмом и авангардом, между Америкой и Европой.

Используя современные технологии машинного обучения, мы воссоздаём художественные стили великих мастеров и применяем их к современным темам, создавая таким образом альтернативную визуальную историю, которая никогда не существовала, но могла бы существовать.

Этот эксперимент показывает не только технические возможности нейросетей, но и универсальность художественного языка, способного выразить любую эпоху через призму своих уникальных средств выразительности.
